# 电影海报日期颜色优化

## Context
电影卡片底部的年份日期使用了 `--color-text-muted` (#555555)，在深色背景 (#12121a) 上对比度很低，不够醒目。需要更换为一个更亮眼的颜色，区别于背景和其他文本。

## 改动

### 修改文件
**`src/components/MovieCard.tsx`** 第 305 行

将日期 span 的 color 从 `var(--color-text-muted)` 改为 `var(--color-gold)`（项目已有的金色高亮色 #d4af37）：

```
- color: 'var(--color-text-muted)'
+ color: 'var(--color-gold)'
```

改动范围：仅一行内联样式属性值。

## 验证
- 启动开发服务器，打开首页
- 确认电影卡片上的年份文字（如 📅 1994）显示为金色 (#d4af37)
- 确认金色在深色卡片背景上对比度足够，易于辨识
- 确认金色与周围的类型标签颜色不冲突
